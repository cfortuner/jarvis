# vision

This package includes all the code for computer vision related tasks, including gaze detection, OCR, object detection, classification, and CAPTCHA solving.

Example layout

```txt
vision/
    gaze_detection/
        eye_tracking.py
        screen_coordinates.py
        utils.py
    ocr/
        offline_ocr.py
        google_ocr.py
        amazon_ocr.py
```
